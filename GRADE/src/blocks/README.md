# Blocos de Conteúdo

Estrutura modular para organizar os slides por blocos temáticos.

## Estrutura

- `00_CAC/` - Bloco 1: Cálcio Coronariano (CAC)
- `01_CALCULADORAS/` - Bloco 2: Calculadoras de Risco
- `02_METAS/` - Bloco 3: Metas de Tratamento
- `03_BEMPEDOICO_SAMS/` - Bloco 4: Bempedoico SAMS

Cada bloco será desenvolvido de forma independente para evitar que alterações em um bloco quebrem os anteriores.
