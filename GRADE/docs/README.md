# GRADE — Aula (Deck HTML)

## Abrir o deck
- Arquivo principal: `GRADE/deck/index.html`
- Recomendado: usar servidor local simples para evitar bloqueios do navegador.

## Estrutura
- `deck/` = apresentação executável
- `docs/` = padrões, checklists e workflow
- `refs/` = PDFs/figuras de referência (quando necessário)
- `archive/legacy-root/` = tudo que existia antes (organizado, mas fora do caminho)

## Regra de ouro
O deck deve continuar abrindo SEM build. Sem frameworks. Simplicidade > complexidade.
