# Project Map

## Pastas
- `deck/`: tudo que o navegador precisa para rodar (HTML + CSS + JS + assets)
- `docs/`: checklists e padrões (qualidade, EBM/GRADE, design, código)
- `refs/`: fontes (PDFs), tabelas e figuras originais
- `archive/legacy-root/`: histórico arquivado (não editar)

## Arquivos principais do deck
- `deck/index.html`: slides
- `deck/css/base.css`: paleta, tipografia, grid, componentes
- `deck/js/navigation.js`: navegação de slides
